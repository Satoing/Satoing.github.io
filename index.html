蚂蚁前端成电行day1

发表于 2021-10-30 更新于 2021-10-31 分类于 大二上

一周前就看到阿里蚂蚁集团的前端大佬到咱学校讲前端的课程报名通知，我磨叽到最后一天才报名。万一哪一天真就准备去阿里实习了呢；；；总之这样把自己说服了。然后呢，该听的课就得认真听讲，对吧。

讲课的安排如下：

我之前已经看到不少对HTML、CSS和JS这样类似的比喻了。大概就是，HTML是网页的框架，CSS对网页的布局和样式进行设定，JavaScrpt则可制作一系列动画。不过，这仅仅是我这种小白的粗浅理解。

下面按照课程的顺序，对当天学习的内容进行梳理。

HTML

HTML全称为HyperText Markup Language。看到这样的名字，一定会想何为超文本，何为标记语言？其间的关系有点复杂，但是我们可以明确，HTML和word这种富文本在布局和文本的组织上都有所不同。

HTML中的标签

  • h标签：标题，包括h1,h2,h3,h4,h5五个层级。
  • p标签：段落标签，为块级元素。
  • a标签：以<a href=""></a>的形式添加链接。
  • img标签：以<img src=" " />的形式引入图片。注意src和href的区别，前者为向文档中引入链接内容，而后者是点击链接跳转。
  • ol标签：有序列表。
  • ul标签：无序列表。
  • li标签：列表项目list item。
  • form标签：表单元素，常用的type有：text、password、submit等。

DOM

DOM即Document Object Moudel,是指从浏览器的视角来观察页面中的元素即每个元素的属性，从而得到一个“家族树”。

它是浏览器文档的模型.

回顾了上面的基础知识，我们写HTML文档应该就没有什么问题了。

CSS

CSS即Cascading Style Sheets，在HTML中引入CSS的时候就使用的是<link rel="stylesheet" href="">。

CSS的原理

CSS通过给文档对象模型中指定的对象添加样式，从而构成页面整体的布局。

而CSS的选择器，就离不开DOM tree。

CSS的语法

css的语法就和之前看得比较多的.json一样，都是键值对的形式。

不同的标签有不同的属性，这点需要我们去熟练和记忆。

引入CSS的四种方式

按照CSS和HTML的分离程度：

  1. 行内样式：直接在标签后面添加样式。
  2. 内联样式：在<head>标签内通过<style>标签给出。这时候要使用选择器。
  3. 外部样式：使用<link rel="stylesheet" href="">导入外部的css文件。
  4. 导入样式：使用@import url;的方式导入外部的css文件。

CSS的选择器

CSS的选择器很多，根据需求选择不同的选择器可以减少样式的重复设定。

但是，为了简便，我们主要使用元素选择器和类选择器。

元素选择器

元素即前面的标签，前面什么都没有。

类选择器

类需要我们在HTML标签后面添加class='name，然后我们就可以在CSS中通过.name选中这个元素。

一个元素可以添加多个类，使用很灵活的。

此外，在css中，我们还可以对类进行分组，即name1,name2,name3……，进行批量操作。

常常有人说，有了div和类选择器就基本上可以完成页面框架的搭建了。对，只不过这是不一种优雅的方式。

CSS 盒模型

它在我们inspect某个网页的时候也见得不少。

为了更精确得设计网页，我们得严格控制padding和margin的宽度。

设置宽度的方式有三种，一是按上下和左右分组，需要两组值；二是按上、右、下、左（顺时针旋转）设置，需要四组值；三是单独设置，比如padding-top:,padding-right:,padding-bottom:,padding-left:。

显然，padding在盒内，称为内距或填充；margin在盒外，称为外距。我们得清楚什么时候设置padding，什么时候又该设置margin：padding控制元素内容和元素内边框之间的距离，而margin控制该元素和相邻元素之间的距离。

最后的问题就是盒子的宽度到底是多少？仅仅是内容的宽度，还是边框以内？

我们来观察一下这个盒子。加一下就会发现，盒子的宽度为border*2+padding-left+width+padding-right，高度同理，没有算上margin。

使用border-radius将得到一个圆形的盒子。

CSS元素格式化

通过display:可设置元素的格式，可选的有：block,inline,inline-block。我们主要研究它的高度：

  • block：元素会撑满一行。
  • inline：元素宽度由内容决定。
  • inline-block：元素宽度默认由内容决定，可自行设定。

CSS背景

我们常用的属性有：background-image,background-position,background-size。

  1. background-image加载的图片链接需要用url()。
  2. background-size用百分比设置图片大小是相对图片原来的大小。由于有时候图片本来的大小并不知道，所以我们一般用px的方式来设置。
  3. background-position有两个值，第一个是水平位置，第二个是垂直位置。我们可以直接给出top、buttom、left、right和center，也可以用百分比，还可以用具体的数值设定背景图位置。
  4. background-repeat一般都会用no-repeat。
  5. 此外还有background-attachment,可以来设定背景是否随滚动条移动，值有fixed，scroll。

CSS定位

PS:定位的值可以是负值。

flex布局

flex是现代流行的一种布局方式：

flex的优点是布局简单，移动端支持比较好。

  1. 父类设置display:flex，即可称为flex容器。其直接子元素将按flex进行布局。
  2. flex布局下，有主轴和侧轴之分。默认主轴为row，侧轴为column。通过flex-direction，可以更改主轴，可选项有row-reverse,column,column-reverse。
  3. 在flex下，我们可以轻松改变子元素的排列方式。通过justify-content进行设定。
  4. 当父元素宽度不够时，子元素默认情况下不会换行，导致挤压变形。这时，我们可以通过flex-wrap:wrap使之换行。配合flex-direct还可以进行多种组合，更体现了flex的灵活性。
  5. 要使元素居中对齐，除了设置主轴外，还需要设置元素在侧轴的排列。使用align-items即可。其参数和justify-content相同。
  6. 上面的属性设置针对的是父类。对于子元素，还可以设置它的顺序order(由小到大，相等并列)，flex-grow等等。

单链表及其基本操作

发表于 2021-10-14 更新于 2021-10-15 分类于 大二上

单链表是最简单确定链表，由一系列节点组成的序列，包括三部分：头指针、头节点、元节点。其中头节点根据实际需求可以不用。有头节点的优点是可保证链表非空，便于判断其他无效的指针即空指针。

单链表节点的定义

单链表的节点包括数据域和指针域。指针域中的指针指向下一个节点；数据域用于存放数据，存放的数据也可以是指针类型。
